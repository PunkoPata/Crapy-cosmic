This is an third party utility for the game Eve Online.

It allows to quickly check which new cosmics signatures appears in the system.

Specially Usefull for wormhole and explorer pilots.

Tested in Windows 11.

== How to use it ==

Execute the program.
In the game select all the content of the Probe Scan Window and Copy them ( Ctr-A Ctr-C )

The program will monitorize your clipboard for new content.
And if it finds differences it will show wich Cosmic Signatures has changed.
